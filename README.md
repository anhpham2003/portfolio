# DSC 106 - Personal Website (lab series)
This project is part of DSC 106 (Data Visualization) and focuses on building a personal website using HTML and CSS.

## Overview
I created a multi-page website that includes a home page, projects page, resume page, and contact page. The goal of this lab was to practice structuring content with semantic HTML and improving layout and design with CSS.

## Features
- Responsive navigation bar using Flexbox  
- Projects page with a dynamic grid layout using CSS Grid  
- Resume page structured with sections and articles  
- Consistent styling and improved visual hierarchy  

## What I Learned
In this lab, I learned how to use Flexbox and CSS Grid for layout, and how small design choices like spacing, alignment, and color can significantly improve the look and usability of a website.
